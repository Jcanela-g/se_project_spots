# Spots

This project is made useing HTML and CSS base on designer details on FIGMA.

**Figma**

## Project features

- Semantic HTML5
- CSS
- Flexbox
- Grid
- Responsive design
- Positioning
- Flat BEM file structure
- Media queries
- Hidden text overflow
